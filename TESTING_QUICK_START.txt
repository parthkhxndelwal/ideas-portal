TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Issue: App doesn't show API Key Setup screen
→ Make sure app directory: cd App
→ Start fresh: npm start
→ Clear Expo cache: npm start --clear

Issue: "API key should start with scanner_"
→ Copy full key from admin dashboard
→ Key format: scanner_xxxxx_xxxxx
→ Don't cut off the "scanner_" prefix

Issue: "Connection Failed" still appears
→ This shouldn't happen with the fix!
→ Check that api-key-setup.tsx was created
→ Check that services.ts has getApiKey/setApiKey
→ Verify API key was entered and saved

Issue: QR code won't decrypt
→ Backend must generate with AES-256-CBC
→ App decryption is in crypto.ts (verified ✅)
→ Should support both new (AES) and old (3-digit)
